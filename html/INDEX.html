✨ Key Features

  • 100% Pure HTML/CSS/JavaScript - No React, No build tools, No dependencies
  • Complete dark/light mode toggle with localStorage persistence
  • Fully responsive design - Mobile, Tablet, Desktop optimized
  • Smooth scroll animations with Intersection Observer
  • Progress bars tracking scroll position on all pages
  • Parallax effects and gradient animations
  • Interactive forms with modals (Contact & Testimonial)
  • Mobile hamburger menu with smooth transitions
  • All navigation links work correctly
  • Back buttons navigate to homepage #work section
  • Optimized performance (~60fps animations)
  • Cross-browser compatible (Chrome, Firefox, Safari, Edge)

🚀 How to Use

  • Option 1: Simply open index.html in your web browser
  • Option 2: Use a local server for best experience (python -m http.server 8000)
  • Option 3: Deploy to any static hosting (Netlify, Vercel, GitHub Pages)
  • All pages are interconnected with proper navigation
  • Theme preference is saved automatically
  • Works offline once loaded (except external images)

⚙️ Technical Details

📦
No Dependencies
Zero external libraries or frameworks. All functionality built with vanilla JavaScript, CSS, and HTML. No npm, no webpack, no compilation step required.
🎨
CSS Animations
All Framer Motion animations converted to CSS @keyframes. Includes fadeIn, fadeInUp, fadeInLeft, scaleIn, bounce, gradientShift, and more.
⚡
Performance
Optimized with Intersection Observer for lazy animations, throttled scroll events, CSS transforms for smooth 60fps animations, and minimal repaints.
📱
Responsive
Mobile-first design with fluid typography using clamp(), flexible grids with CSS Grid, and breakpoints at 640px, 768px, and 1024px.
